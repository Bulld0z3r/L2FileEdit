##L2FileEdit
适用于编辑天堂2客户端system内文件<br>
For LineageII client files in system folder

-------------------
##功能

* 编辑system内文件(dat, ini, int)<br>
* 自动加解密dat文档(使用1datpatch.bat, 先把dat放在data目录下)
